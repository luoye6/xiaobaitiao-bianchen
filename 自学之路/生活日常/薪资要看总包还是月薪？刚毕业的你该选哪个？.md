# 薪资要看总包还是月薪？刚毕业的你该选哪个？

> 本文作者：[程序员小白条](https://github.com/luoye6)
>
> 本站地址：[https://xbt.xiaobaitiao.top](https://xbt.xiaobaitiao.top)

每次看到offer上的薪资数字，都像在解一道谜题：月薪、年终奖、股票、补贴...到底该信哪个？哪个才不会坑我？

刚毕业的我们，谈薪时就像在菜市场买菜——既怕被坑，又怕错过好货。HR说的"年薪30万"听着很香，但仔细一看，月薪才1万，剩下全靠年终奖和股票。

这感觉就像买奶茶，说好的大杯，结果半杯都是冰块！

我建议还是多去问清楚，薪资结构以及总包具体组成部分，对于一些不稳定的还是要慎重，极大概率不会写在劳动合同里，因此不发也是基操，毕竟仲裁也没用。

### 一、月薪vs总包，到底该信谁？

月薪毕竟每一个就得发一次，五险一金也是如此，你每次跳槽都是满X年整或者拿年终再跑路的？有时候很难自己去决定，环境和外界因素会影响你！

**月薪党：落袋为安才是真**

支持月薪的同学理由很实在：

- 月薪是每个月实实在在到手的钱，不像年终奖可能要等第二年
- 五险一金都按月薪算，月薪高公积金也高
- 跳槽时下家公司都按你之前的月薪来谈涨幅
- 绩效？年终奖？那都是薛定谔的钱——可能有，也可能没有

有位网友说得特别扎心："月薪两万，年薪大概率30万以上；但年薪30万，月薪可能只有一万，剩下的都得看公司脸色。"

**总包党：眼光要放长远**

支持总包的同学则认为：

- 有些公司年终奖真的很多，比如某东最高能到20薪
- 股票期权万一升值了，那可是意外之喜
- 各种补贴加起来也不是小数目
- 公积金也是钱啊，而且能取出来用

### 二、识破薪资里的"水分"

年终奖就是额外福利，发不发属于都是属于公司决定的，很多还要根据你绩效评定的等级进行发，另外公司盈利如何也是一大因素，因此给你说什么16薪，20薪，你得看到时候发不发啊，假如月base因此变的很低，反而给高的年终，这就要好好考虑了，很多会延迟好久才发，就是为了那些早离职的人就可以不用发这些奖金了。

**年终奖的套路**

HR说"年薪包含4个月年终奖"，但要问清楚：

- 是所有人都能拿满，还是只有绩效好的才能拿？
- 过去三年实际发放比例是多少？
- 如果中途离职，还能拿吗？

**股票的陷阱**

"年薪包含价值10万的股票"——听着很美好，但要确认：

- 分几年给？每年给多少？
- 离职了还没给的股票怎么办？
- 行权价格是多少？别到时候还要自己掏钱买

**公积金的门道**

同样月薪2万，公积金按12%交和按5%交，一年能差出好几万！这可是真金白银的差别。

### 三、谈薪时一定要问清楚这些

**必问清单：**

1. 月薪具体多少？每个月几号发？
2. 五险一金按什么基数和比例交？
3. 年终奖一般能拿几个月？怎么考核？
4. 有没有其他补贴？（餐补、交通补、房补）
5. 上班时间？加班有没有加班费？

**避坑指南：**

- 用"offer工资计算器"小程序算算实际到手
- 问问在职的学长学姐，实际能拿多少
- 警惕"画大饼"式的薪资结构

------

说好的年终奖可能缩水，但每个月按时到账的工资不会骗人。

**对于刚毕业的我们，稳定的高月薪可能比虚无缥缈的总包更实在。**

等在职场上混出名堂了，再去追求那些带着股票期权的大礼包也不迟！记住，谈薪时脸皮要厚，该问的一定要问清楚！
